🛠️ 真正从零开始的保姆级 SOP

机房 VPS + 住宅代理 + Hysteria2 + sing-box 完整架构

🧱 第一阶段：准备阶段

从购买基础设施开始，这是你所有网络流量的基石。

1 购买机房 VPS

推荐 Ubuntu 22.04 系统。这台服务器作为流量中转站。

👉 前往购买机房 VPS

2 购买住宅代理

住宅 IP 是防封的关键，它会让你的出口看起来像普通家庭用户。

👉 前往购买住宅代理

3 下载 SSH 工具

使用 FinalShell 连接你的 VPS 黑色终端。

点击下载 FinalShell

🧩 第二阶段：服务器逐行部署

请打开 FinalShell，连接上 VPS，然后**一行一行**执行下面的命令。

1 更新系统环境

apt update && apt upgrade -y

2 安装必备基础工具

apt install -y curl openssl jq

3 安装 sing-box (官方全自动方式)

bash <(curl -fsSL https://sing-box.app/install.sh)

4 生成自签 TLS 证书

mkdir -p /etc/sing-box && openssl req -x509 -nodes -days 3650 -newkey rsa:2048 -keyout /etc/sing-box/private.key -out /etc/sing-box/cert.pem -subj "/CN=www.bing.com"

5 编辑配置文件

先输入以下命令打开编辑器：

nano /etc/sing-box/config.json

打开后，删光里面所有内容。然后复制下面这段 JSON 粘贴进去：

💡 记得把 住宅IP, 端口, 用户名, 密码 改成你自己的。
{
  "log": { "level": "info" },
  "inbounds": [
    {
      "type": "hysteria2",
      "listen": "::",
      "listen_port": 42292,
      "users": [ { "password": "hy2password" } ],
      "tls": {
        "enabled": true,
        "certificate_path": "/etc/sing-box/cert.pem",
        "key_path": "/etc/sing-box/private.key"
      }
    }
  ],
  "outbounds": [
    {
      "type": "http",
      "tag": "webshare",
      "server": "你的住宅代理IP",
      "server_port": 你的住宅端口,
      "username": "你的用户名",
      "password": "你的密码"
    },
    { "type": "direct", "tag": "direct" }
  ],
  "route": { "final": "webshare" }
}

(保存方法：按 Ctrl+O，回车，再按 Ctrl+X)

6 启动服务并设置自启动

systemctl restart sing-box && systemctl enable sing-box

7 检查运行状态

ss -lunp | grep 42292

如果看到 UNCONN ... :42292 样式的输出，说明成功！

📱 第三阶段：客户端一键使用

iOS (小火箭 Shadowrocket)

手动添加类型为 Hysteria2 的节点：

  • 服务器：你的VPS_IP
  • 端口：42292
  • 密码：hy2password
  • 允许不安全 (Insecure)：开启 (因为是自签证书)
  • SNI：www.bing.com

PC (Clash Verge)

在配置文件 proxies 下添加：

- name: "My-Hy2-Residential"
  type: hysteria2
  server: 你的VPS_IP
  port: 42292
  password: hy2password
  sni: www.bing.com
  skip-cert-verify: true
  alpn: [h3]
