🚀 机房 + 住宅 套壳量产 SOP

适用于：VPS + Webshare 住宅 + iPhone 小火箭

第一阶段：准备材料

① 机房 VPS

  • 美国机房
  • Ubuntu 22.04
  • Root 权限

② Webshare 住宅

记录： 

住宅IP:
端口:
用户名:
密码:

③ SSH 工具

推荐：FinalShell

第二阶段：安装 sing-box


wget https://github.com/SagerNet/sing-box/releases/download/v1.13.9/sing-box_1.13.9_linux_amd64.deb
dpkg -i sing-box_1.13.9_linux_amd64.deb

第三阶段：标准配置模板


{
  "log": {
    "level": "info"
  },

  "inbounds": [
    {
      "type": "hysteria2",
      "tag": "hy2",
      "listen": "0.0.0.0",
      "listen_port": 42293,
      "users": [
        {
          "password": "你的密码"
        }
      ],
      "tls": {
        "enabled": false
      }
    }
  ],

  "outbounds": [
    {
      "type": "http",
      "tag": "webshare",
      "server": "住宅IP",
      "server_port": 住宅端口,
      "username": "住宅用户名",
      "password": "住宅密码"
    }
  ],

  "route": {
    "final": "webshare"
  }
}

第四阶段：启动服务


systemctl restart sing-box
systemctl status sing-box
ss -lunp | grep 42293

第五阶段：生成小火箭导入链接

格式： 

hysteria2://密码@VPSIP:42293?sni=www.bing.com&insecure=1#Webshare-住宅
