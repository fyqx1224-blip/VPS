🚀 机房 + 住宅 套壳量产保姆级 SOP

第一步：购买机房 VPS

推荐使用 Racknerd（支持支付宝 / 微信支付）

👉 点击购买 VPS

推荐配置：
• 1核 1G 起步
• Ubuntu 22.04
• 美国机房

第二步：购买住宅代理

推荐使用 Webshare 住宅

👉 点击注册 Webshare

购买后记录：
住宅IP / 端口 / 用户名 / 密码

第三步：下载 SSH 工具

推荐 FinalShell

👉 下载 FinalShell

安装后：
新建 SSH → 填写 VPS IP → root → 密码 → 连接

第四步：安装 sing-box


wget https://github.com/SagerNet/sing-box/releases/download/v1.13.9/sing-box_1.13.9_linux_amd64.deb
dpkg -i sing-box_1.13.9_linux_amd64.deb

第五步：写入标准配置


{
  "log": { "level": "info" },

  "inbounds": [
    {
      "type": "hysteria2",
      "tag": "hy2",
      "listen": "0.0.0.0",
      "listen_port": 42293,
      "users": [
        { "password": "自定义密码" }
      ],
      "tls": { "enabled": false }
    }
  ],

  "outbounds": [
    {
      "type": "http",
      "tag": "webshare",
      "server": "住宅IP",
      "server_port": 住宅端口,
      "username": "住宅用户名",
      "password": "住宅密码"
    }
  ],

  "route": {
    "final": "webshare"
  }
}

第六步：启动服务


systemctl restart sing-box
systemctl status sing-box
ss -lunp | grep 42293

第七步：iOS 小火箭导入

生成链接格式： 

hysteria2://密码@VPSIP:42293?sni=www.bing.com&insecure=1#住宅
在 Safari 打开即可自动跳转导入小火箭。

第八步：PC 使用 Clash Verge

👉 下载 Clash Verge

新建配置 → 粘贴 YAML → 启动
