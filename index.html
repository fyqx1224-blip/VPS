🚀 从零开始的科学上网配置指南

基于 VPS + 住宅代理 + Hysteria2 + sing-box 的极致防封抗封锁方案

🧱 第一阶段：准备工作

你需要准备一台海外服务器（VPS）和一个住宅代理。这是保证节点既快又不容易被封的核心。

① 购买机房 VPS

  • 推荐配置：1核 1G 内存以上，1Gbps 带宽。
  • 推荐区域：洛杉矶 / 日本 / 新加坡。
  • 系统务必选择：Ubuntu 22.04
👉 点击前往 RackNerd 购买

② 购买住宅代理

用于将你的最终 IP 伪装成真实的家庭宽带。

👉 点击前往 Webshare 购买
⚠️ 购买后请记下：住宅 IP、端口、用户名、密码。稍后配置时需要用到。

③ 下载 SSH 连接工具

推荐使用 FinalShell，用于连接和控制你的 VPS。

下载 FinalShell

⚡ 第二阶段：一键部署核心服务

打开 FinalShell，连接上你的 VPS（输入 IP、用户名 root、及密码）。进入黑色命令行界面后，直接复制并粘贴下方的一键部署脚本，然后回车执行。

cat << 'EOF' > install_hy2_proxy.sh
#!/bin/bash

echo "=========================================="
echo "🚀 开始一键部署 sing-box Hysteria2 节点..."
echo "=========================================="

echo "📦 正在更新系统并安装依赖..."
apt update && apt upgrade -y
apt install -y curl openssl jq

echo "📦 正在安装 sing-box..."
bash <(curl -fsSL https://sing-box.app/install.sh)

echo "🔐 正在生成自签证书..."
mkdir -p /etc/sing-box
openssl req -x509 -nodes -days 3650 -newkey rsa:2048 \
-keyout /etc/sing-box/private.key \
-out /etc/sing-box/cert.pem \
-subj "/CN=www.bing.com" 2>/dev/null

echo "=========================================="
echo "📝 请输入你的住宅代理信息："
read -p "住宅代理 IP/域名: " proxy_ip
read -p "住宅代理 端口: " proxy_port
read -p "住宅代理 用户名: " proxy_user
read -p "住宅代理 密码: " proxy_pass
read -p "请设置 Hysteria2 连接密码 (直接回车默认: hy2password): " hy2_pass
hy2_pass=${hy2_pass:-hy2password}

echo "⚙️ 正在生成 sing-box 配置文件..."
cat > /etc/sing-box/config.json <<EOL
{
  "log": { "level": "info" },
  "inbounds": [
    {
      "type": "hysteria2",
      "listen": "::",
      "listen_port": 42292,
      "users": [ { "password": "$hy2_pass" } ],
      "tls": {
        "enabled": true,
        "certificate_path": "/etc/sing-box/cert.pem",
        "key_path": "/etc/sing-box/private.key"
      }
    }
  ],
  "outbounds": [
    {
      "type": "http",
      "tag": "webshare",
      "server": "$proxy_ip",
      "server_port": $proxy_port,
      "username": "$proxy_user",
      "password": "$proxy_pass"
    },
    { "type": "direct", "tag": "direct" }
  ],
  "route": { "final": "webshare" }
}
EOL

echo "🔄 重启 sing-box 服务..."
systemctl restart sing-box
systemctl enable sing-box

server_ip=$(curl -s ipv4.icanhazip.com)

echo "=========================================="
echo "✅ 部署完成！"
echo "=========================================="
echo ""
echo "📱 【iOS 小火箭 / Shadowrocket 一键导入链接】"
echo "hysteria2://$hy2_pass@$server_ip:42292?sni=www.bing.com&insecure=1&obfs=none#Hy2-住宅节点"
echo ""
echo "💻 【PC Clash Verge 专属配置片段】"
cat <<EOL
proxies:
  - name: Hy2-Node
    type: hysteria2
    server: $server_ip
    port: 42292
    password: $hy2_pass
    sni: www.bing.com
    skip-cert-verify: true
    alpn:
      - h3
EOL
echo "=========================================="
EOF

bash install_hy2_proxy.sh

脚本运行过程中，会提示你输入刚才记下的「住宅代理」信息。按提示输入并回车即可。运行结束后，终端会直接打印出客户端需要的配置信息！

📱 第三阶段：客户端连接

iOS 用户 (Shadowrocket / 小火箭)

直接复制脚本运行完毕后生成的 hysteria2://... 链接，打开小火箭，它会自动识别并弹窗提示添加。

PC 用户 (Clash Verge)

打开 Clash Verge，新建一个本地配置文件，将脚本生成的 proxies 及之后的代码段复制粘贴进去即可使用。
